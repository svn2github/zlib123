OpenXML/ODF Translator Add-in for Microsoft Powerpoint (Microsoft Powerpoint 用 OpenXML/ODF 変換アドイン プログラム)

ODF Add-in for Excel XP (Excel XP 用 ODF アドイン プログラム) をダウンロードいただき、ありがとうございます。このアドインをご利用いただくことにより Microsoft Powerpoint の ODF 仕様に基づいてドキュメントを開き、Office Open XML と ODF の間の相互運用性を活用することができます。この相互運用性プロジェクトは、sourceforge.netがホストしています。プロジェクトの詳細および今後の更新については、ロードマップ ドキュメントをご覧ください。

OpenXML/ODF Translator は Microsoft が資金提供し、Sonata Software Limited が開発しました (詳しくは、プロジェクト Web サイト をご覧ください)。

ソフトウェアの要件

このアドイン プログラムをテストするには、以下の要件を満たしていただく必要があります。

  • Microsoft Powerpoint XP
  • Microsoft Office ファイル形式互換機能パック
  • .NET Framework 2.0 SP1

または

  • Microsoft Powerpoint 2003
  • Microsoft Office ファイル形式互換機能パック
  • .NET Framework 2.0* SP1

または

  • Microsoft Powerpoint 2007 (.NET Programmability Support が有効であること)
  • .NET Framework 2.0* SP1

(*) 注:Microsoft Powerpoint が既にインストールされている状態で .Net Framework をインストールした場合は、次の手順に従って Powerpoint を更新してください。

  1. [コントロール パネル] の [プログラムの追加と削除] をダブルクリックします。
  2. Microsoft Powerpoint を探し、[変更] をクリックします。
  3. インストール プログラムの [機能の追加と削除] を選択し、[Microsoft Office Powerpoint の .NET プログラミング サポート] を追加します。

アドインのインストール

OdfAddInForPowerpointSetup-en.exe をダブルクリックします。

インストールが完了すると、Powerpoint で .ODP ファイル (.ODP は DF テキスト ファイルの標準拡張子) が開けるようになります ([ファイル] メニューの [ODF...])。更に、エクスプローラで .ODP ファイルを右クリックすると、そのファイルを Microsoft Powerpoint で開けるオプション (Powerpoint で開く) が表示されます。

重要事項: アドインが開いた ODP ファイルは、Microsoft Powerpoint の読み取り専用の PPTX ファイル形式です。変更したファイルを ODP ファイル形式で保存し直す場合は、[ファイル] メニューの [名前を付けて保存] を選択して、そのドキュメントをいったん保存してから、[ODF 形式で保存] を選択します。

トラブルシューティング

Microsoft Powerpoint の [ファイル] メニューに ODF が表示されない場合は、Powerpoint の必須機能がインストールされていないか (上記の「ソフトウェアの要件」をご覧ください)、または Powerpoint プライマリ相互運用機能アセンブリが正常にインストールされなかった可能性があります。

詳細については、トラブルシューティング ガイド を参照してください。
