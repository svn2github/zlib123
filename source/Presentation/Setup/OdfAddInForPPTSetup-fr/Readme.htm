Complément OpenXML/ODF Translator pour Microsoft Powerpoint

Merci d'avoir téléchargé le complément OpenXML/ODF pour Powerpoint. Ce complément vous permet d'ouvrir, dans Microsoft Powerpoint, des documents basés sur les spécifications ODF et de démontrer l'interfonctionnement entre Office Open XML et ODF. Ce projet d'interfonctionnement est hébergé sur le site sourceforge.net. Pour plus de détails et pour des informations sur les futures mises à jour concernant le projet, veuillez consulter notre feuille de route (RoadMap).

Financé par Microsoft, le projet de traduction OpenXML/ODF Translator a été développé par Sonata Software Limited (pour plus d'informations, veuillez visiter le site Web relatif au projet).

Configuration logicielle requise

Avant d'installer le complément, vérifiez que vous disposez de l'un des éléments suivants...

  • Microsoft Powerpoint XP
  • Module de compatibilité Office
  • .NET Framework 2.0 SP1

ou

  • Microsoft Powerpoint 2003
  • Module de compatibilité Office
  • .NET Framework 2.0* SP1

ou

  • Microsoft Powerpoint 2007 (avec l'option Prise en charge de la programmabilité .NET activée)
  • .NET Framework 2.0* SP1

(*) Remarque : Si vous avez installé Microsoft Powerpoint avant .NET Framework, vous devez mettre à jour votre installation de Powerpoint comme suit :

  1. Ouvrez Ajout/Suppression de programmes dans le Panneau de configuration.
  2. Recherchez l'entrée Microsoft Powerpoint et choisissez Modifier.
  3. Au cours du processus d'installation, choisissez Ajouter ou supprimer des composants puis ajoutez Prise en charge de la programmabilité .NET dans la section Microsoft Office Powerpoint.

Installation du complément

Effectuez un double clic sur le fichier « OdfAddInForPowerpointSetup-en.exe »

Si l'installation réussit, vous pouvez ouvrir un fichier .ODP (ODP est l'extension de fichier standard des documents ODF) dans Microsoft Powerpoint (voir le menu Fichier>ODF...). En outre, lorsque vous effectuez un clic droit sur un fichier .ODP dans l'Explorateur, vous devez pouvoir ouvrir le document dans Microsoft Powerpoint (menu Ouvrir dans Powerpoint).

Remarque importante : Les fichiers ODP ouverts par le complément sont importés dans Microsoft Powerpoint en tant que fichiers PPTX en lecture seule. Par conséquent, si vous avez apporté des modifications à un fichier et que vous l'avez enregistré en tant que fichier .ODP, vous devez d'abord l'enregistrer sous un nouveau nom (Fichier>Enregistrer sous), puis utiliser la commande Fichier>ODF…-Enregistrer sous ODF….

Dépannage

Si l'entrée ODF n'apparaît pas dans le menu Fichier de Microsoft Powerpoint, il est possible qu'il manque des fonctionnalités dans Powerpoint (voir la section Configuration logicielle requise) ou qu'une panne se soit produite lors de l'installation des assemblys PIA (Primary Interop Assemblies) de Powerpoint.

Veuillez consulter notre Guide de dépannage pour plus d'informations.
