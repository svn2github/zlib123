OpenXML/ODF Translator Add-In für Microsoft Powerpoint

Herzlichen Dank, dass Sie das OpenXML/ODF Translator Add-In für Powerpoint heruntergeladen haben. Mit diesem Add-In können Sie Dokumente, die auf der Open Document Format (ODF) Spezifikation basieren, in Microsoft Powerpoint öffnen bzw. mit Word erstellte Dokumente in diesem Format speichern. Damit ist die Interoperabilität zwischen den Spezifikationen OpenXML (Microsoft Powerpoint) und ODF gewährleistet. Das Projekt OpenXML/ODF Translator Add-In für Microsoft Powerpoint wird auf sourceforge.net. gehosted. Weitere Einzelheiten und zukünftige Updates entnehmen Sie bitte dort dem Dokument "RoadMap".

Der OpenXML/ODF Translator wird von Microsoft finanziell unterstützt und von Sonata Software Limited entwickelt (mehr Informationen darüber finden Sie hier).

Software-Voraussetzungen

Bevor Sie das Add-In installieren, vergewissern Sie sich, dass folgende Software installiert ist ...

  • Microsoft Powerpoint XP
  • Office Compatibility Pack
  • .NET Framework 2.0 SP1

oder

  • Microsoft Powerpoint 2003
  • Office Compatibility Pack
  • .NET Framework 2.0 SP1

oder

  • Microsoft Powerpoint 2007 (mit aktivierter .NET-Programmierunterstützung)
  • .NET Framework 2.0* SP1

(*) Hinweis: Wenn Sie Microsoft Word vor dem .NET Framework installiert haben, müssen Sie Ihre Powerpoint-Installation aktualisieren. Dies geschieht folgendermaßen.

  1. Öffnen Sie Software in der Systemsteuerung
  2. Suchen Sie den Microsoft Powerpoint Eintrag und wählen Sie Ändern
  3. Während des Installationsprozesses wählen Sie Features ändern oder entfernen und fügen Sie .NET-Programmierunterstützung in der Microsoft Office Powerpoint Sektion hinzu.

Das Add-In installieren

Machen Sie einen Doppelklick auf "OdfAddInForPowerpointSetup-de.exe".

Nach erfolgreicher Installation können Sie .ODP Dokumente (das ist die Dateierweiterung für ODF Präsentationsdokumente) in Microsoft Powerpoint öffnen (siehe Kommando Datei>ODF). Auch können Sie .ODP Dokumente im Explorer durch Klick auf die rechte Maustaste mit Microsoft Powerpoint öffnen (Kommando mit Powerpoint öffnen).

Wichtige Anmerkung: Das .ODP Dokument wird als schreibgeschütztes .PPTX Dokument in Microsoft Powerpoint geöffnet. Wenn Sie das Dokument verändert haben und es wieder als .ODP Dokument speichern wollen, müssen Sie das Dokument zuerst unter einem anderen Namen speichern (Datei>Speichern unter) und danach das Kommando Datei>Speichern als ODF aufrufen.

Beseitigen von Problemen

Wenn Sie den ODF Eintrag im Microsoft Powerpoint Menü nicht finden, kann dies an einer fehlenden Powerpoint-Komponente (siehe auch Software-Voraussetzungen) oder an einer fehlerhaften Installation der "Powerpoint Primary Interop Assemblies" liegen.

Mehr Informationen darüber finden Sie in unserem Troubleshooting Guide.
