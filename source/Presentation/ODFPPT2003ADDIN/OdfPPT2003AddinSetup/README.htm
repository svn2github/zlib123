ODF Add-in for PowerPoint 2003

Thank you for downloading the ODF Add-in PowerPoint 2003. The purpose of this software is to enable the user to open an ODF Specification based document in PowerPoint 2003 and demonstrate the interoperability between OpenXML and ODF specifications. Since, the Add-in is still a beta release; nevertheless you may use this to test it at your own risk! However, the feedback is welcome though!

Software Requirements

Before installing ODF Add-In for PowerPoint make sure to install all the High- Priority updates from Microsoft Update.

To test the add-in, you will need the following:
1. PowerPoint 2003
2. .NET Framework 2.0*
3. Compatibility Pack* for Word, Excel, and PowerPoint 2007 File Formats (beta 2 or higher).

IMPORTANT NOTES:

(*) Warning: If Microsoft PowerPoint is installed before .Net Framework, you need to update the PowerPoint by executing/ running the steps below.

1. From Control Panel Double-click Add/Remove Programs.
2. In the list of currently installed programs find for Microsoft PowerPoint, choose Modify.
3. Within the installation process choose Add/Remove features, add .Net Programmability Support in Microsoft Office PowerPoint.

Installing the add-in

Follow the steps to install the downloaded program:

Once the installation is complete, you can notice that in the PowerPoint, File Menu a new menu item, Open ODF is added. It allows either importing the ODF PowerPoint file or exporting the current working document to an ODF PowerPoint file (Note: During the development process these functionalities may be temporarily unavailable).

Important note: An ODF file opened by the add-in is imported in Microsoft PowerPoint as a read-only PPTX file. However, if you want to save it back to ODF file, first make a copy of this document ("Menu>Save As") and then use “Export as ODF".

Troubleshooting Guide

If the ODF entry is not located, it can be due to the missing features in PowerPoint (Look up for the Software Requirements listed above). After completing the ODF Add-in installation, you need to activate the Add-in by the following actions:

* On the File Menu, click PowerPoint Options.
* In the Add-ins section, select COM Add-ins in the list box, press Go button.
* Make sure to select the checkbox ODF PowerPoint 2003 Add-in and validate with the OK button.
