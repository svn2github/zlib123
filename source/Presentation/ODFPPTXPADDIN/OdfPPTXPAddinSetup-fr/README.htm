Complément ODF pour PowerPoint XP

Merci d’avoir téléchargé le complément ODF pour PowerPoint XP. Grâce à ce logiciel, l’utilisateur peut ouvrir dans PowerPoint XP un document basé sur les spécifications ODF et démontrer l’interfonctionnement entre les spécifications OpenXML et ODF. Bien que le complément soit encore en version bêta, vous pouvez l’utiliser à vos propres risques, à des fins de test. N’hésitez pas à nous envoyer vos commentaires !

Configuration logicielle requise

Avant d’installer le complément ODF pour PowerPoint, prenez soin d’installer toutes les mises à jour à priorité élevée, disponibles sur le site Microsoft Update.

To test the add-in, you will need the following:
1. PowerPoint XP
2. .NET Framework 2.0*
3. Module de compatilibité* pour les formats de fichiers Word, Excel et PowerPoint 2007 (bêta 2 ou version ultérieure).

REMARQUES IMPORTANTES:

(*) Avertissement : Si Microsoft PowerPoint est installé avant .Net Framework, vous devez mettre à jour PowerPoint en effectuant les étapes suivantes.

1. Dans le Panneau de configuration effectuez un double clic sur Ajout/Suppression de programmes.
 2. Dans la liste des programmes actuellement installés, recherchez Microsoft PowerPoint, et choisissez Modifier.
3. Au cours du processus d’installation, choisissez Ajouter/supprimer des composants, et ajoutez Prise en charge de la programmabilité .NET dans Microsoft Office PowerPoint.

Installation du complément

Suivez ces étapes pour installer le programme téléchargé:

Une fois l’installation terminée, vous pouvez remarquer dans PowerPoint qu’un nouvel élément, Open ODF a été ajouté au menu Fichier. Il permet d’importer le fichier PowerPoint ODF ou d’exporter le document de travail en cours vers un fichier PowerPoint ODF.
(Remarque: Au cours du processus de développement, ces fonctionnalités risquent de ne pas être disponibles.

Remarque importante : Un fichier ODF ouvert par le complément est importé dans Microsoft PowerPoint en tant que fichier PPTX en lecture seule. Toutefois, si vous voulez enregistrer à nouveau ce document en tant que fichier ODF, effectuez-en d’abord une copie (« Menu>Enregistrer sous ») puis utilisez « Enregistrer sous ODF ».

Guide de dépannage

L’incapacité à localiser l’entrée ODF peut être due au fait qu’il manque des fonctionnalités dans PowerPoint (consulter la rubrique Configuration logicielle requise, plus haut dans le présent document). Après l’installation du Complément ODF, vous devrez activer le Complément en effectuant les actions suivantes :

* Dans le menu Fichier, cliquez sur Options PowerPoint.
 * Dans la section Compléments , sélectionnez Compléments COM dans la zone de liste, puis cliquez sur le bouton OK.
* Prenez soin d’activer la case à cocher Complément ODF pour PowerPoint XP et de valider votre choix en cliquant sur OK.

Notes de mise à jour

(1) Le Complément ne fonctionne pas lorsque Office XP est installé en parallèle avec Office 2007.
(2) Il est impossible de convertir un fichier PPT. Une telle conversion n'est possible que si l'utilisateur enregistre le fichier PPT sous le format PPTX.
