Complément ODF pour PowerPoint 2007

Merci d’avoir téléchargé le complément ODF pour PowerPoint 2007. Grâce à ce logiciel, l’utilisateur peut ouvrir dans PowerPoint 2007 un document basé sur les spécifications ODF et démontrer l’interfonctionnement entre les spécifications Open XML et ODF.

Configuration logicielle requise

1. PowerPoint 2007
2. .NET Framework 2.0*

REMARQUES IMPORTANTES:

(*) Avertissement : Si Microsoft PowerPoint est installé avant .Net Framework, vous devez mettre à jour PowerPoint en effectuant les étapes suivantes.

1. Dans le Panneau de configuration effectuez un double clic sur Ajout/Suppression de programmes.
2. Dans la liste des programmes actuellement installés, recherchez Microsoft PowerPoint, et choisissez Modifier.
3. Au cours du processus d’installation, choisissez Ajouter/supprimer des composants, et ajoutez Prise en charge de la programmabilité .NET dans Microsoft Office PowerPoint.

Installation du complément

Suivez ces étapes pour installer le programme téléchargé :

Une fois l’installation terminée, vous pouvez remarquer dans PowerPoint qu’un nouvel élément, Open ODF, a été ajouté au menu Fichier. Il permet d’importer le fichier PowerPoint ODF ou d’exporter le document de travail en cours vers un fichier PowerPoint ODF (Remarque: Au cours du processus de développement, ces fonctionnalités risquent de ne pas être disponibles.)

Remarque importante: Un fichier ODF ouvert par le complément est importé dans Microsoft PowerPoint en tant que fichier PPTX en lecture seule. Toutefois, si vous voulez enregistrer à nouveau ce document en tant que fichier ODF, effectuez-en d’abord une copie (« Menu>Enregistrer sous »), puis utilisez « Enregistrer sous ODF ».

Guide de dépannage

L’incapacité à localiser l’entrée ODF peut être due au fait qu’il manque des fonctionnalités dans PowerPoint (consulter la rubrique Configuration logicielle requise, plus haut dans le présent document) ou qu'une panne se soit produite lors de l'installation des assemblys PIA (Primary Interop Assemblies) de PowerPoint.

Veuillez consulter notre Guide de dépannagepour plus d'informations
