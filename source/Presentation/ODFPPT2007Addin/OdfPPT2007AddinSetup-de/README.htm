ODF Add-In für PowerPoint 2007

Vielen Dank für das Herunterladen des ODF Add-Ins für PowerPoint 2007. Die Software ermöglicht das Öffnen eines auf der ODF-Spezifikation basierenden Dokuments in PowerPoint 2007 und die Interoperabilität zwischen OpenXML und ODF-Spezifikationen.

Softwareanforderungen

To test the add-in, you will need the following:
1. PowerPoint 2007
2. .NET Framework 2.0*

WICHTIGE HINWEISE:

(*) Warnung: Wenn Microsoft PowerPoint vor .Net Framework installiert wurde, führen Sie folgende Schritte aus, um PowerPoint zu aktualisieren.

1. Doppelklicken Sie in der Systemsteuerung auf Software.
2. Wählen Sie in der Liste der installierten Programme Microsoft PowerPoint, aus und klicken Sie auf Ändern.
3. Wählen Sie Features hinzufügen/entfernen, aus und fügen Sie die .Net-Programmierunterstützung für Microsoft Office PowerPoint hinzu.

Installieren des Add-Ins

Führen Sie folgende Schritte aus, um das heruntergeladene Programm zu installieren:

Nachdem die Installation abgeschlossen ist, wird im PowerPoint-Menü Datei das neue Menüelement ODF öffnen angezeigt. Mit dieser Option können Sie die ODF PowerPoint-Datei importieren oder das aktuelle Dokument in eine ODF PowerPoint-Datei exportieren. (Hinweis: Während der Entwicklungsphase sind diese Funktionen möglicherweise vorübergehend nicht verfügbar.)

Wichtiger Hinweis: Eine mit dem Add-In geöffnete ODF-Datei wird als schreibgeschützte PPTX-Datei in Microsoft PowerPoint importiert. Um die Datei wieder als ODF-Datei zu speichern, erstellen Sie eine Kopie des Dokuments („Datei > Speichern unter“) und wählen Sie die Option Als ODF exportieren aus.

Problembehandlung

Wenn die ODF-Option nicht vorhanden ist, sind einige Features in PowerPoint möglicherweise nicht verfügbar (siehe Softwareanforderungen). Nachdem die Installation des ODF Add-ins abgeschlossen ist, gehen Sie wie folgt vor, um das Add-In zu aktivieren:

* Klicken Sie im Menü Datei auf PowerPoint-Optionen. * Wählen Sie unter Add-Ins im Listenfeld die Option COM Add-Ins aus und klicken Sie auf Weiter. * Aktivieren Sie das Kontrollkästchen ODF PowerPoint 2007 Add-In und klicken Sie auf OK.
