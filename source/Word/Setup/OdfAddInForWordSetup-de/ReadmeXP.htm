ODF Add-In für Microsoft Word

Herzlichen Dank, dass Sie das ODF Add-In für Word heruntergeladen haben. Mit dieser Software können Sie Dokumente, die auf der Open Document Format (ODF) Spezifikation basieren, in Microsoft Word öffnen bzw. mit Word erstellte Dokumente in diesem Format speichern. Damit ist die Interoperabilität zwischen den Spezifikationen OpenXML (Microsoft Word) und ODF gewährleistet. Das Projekt ODF Add-In für Microsoft Word wird auf sourceforge.net. gehosted. Weitere Einzelheiten und zukünftige Updates entnehmen Sie bitte dort dem Dokument "RoadMap".

Dieses Add-In wurde von Microsoft finanziert und von Clever Age, DIaLOGIKa und Aztecsoft entwickelt (weitere Informationen finden Sie auf der Projekt-Webseite).

Software-Voraussetzungen

Bevor Sie das Add-In installieren, vergewissern Sie sich, dass folgende Software installiert ist ...

  • Microsoft Word XP
  • Office Compatibility Pack
  • .NET framework 2.0

oder

  • Microsoft Word 2003
  • Office Compatibility Pack
  • .NET framework 2.0*

oder

  • Word 2007 mit aktivierter .NET-Programmierunterstützung
  • .NET framework 2.0*

(*) Warnung: Wenn Sie Microsoft Word vor dem .NET Framework installiert haben, müssen Sie Ihre Word-Installation aktualisieren. Dies geschieht folgendermaßen.

  1. Öffnen Sie "Software" in der Systemsteuerung
  2. Suchen Sie den "Microsoft Word" Eintrag und wählen Sie "Ändern"
  3. Während des Installationsprozesses wählen Sie "Features ändern oder entfernen" und fügen Sie ".NET-Programmierunterstützung" in der "Microsoft Office Word" Sektion hinzu.

Das Add-In installieren

Machen Sie einen Doppelklick auf "OdfAddInForWordSetup-de-1.0.exe".

Nach erfolgreicher Installation können Sie .ODT Dokumente (das ist die Dateierweiterung für ODF Text Dokumente) in Microsoft Word öffnen (siehe Kommando "Datei>ODF"). Auch können Sie .ODT Dokumente im Explorer durch Klick auf die rechte Maustaste mit Microsoft Word öffnen (Kommando "mit Word öffnen").

Wichtige Anmerkung: Das .ODT Dokument wird als schreibgeschütztes .DOCX Dokument in Microsoft Word geöffnet. Wenn Sie das Dokument wieder als .ODT Dokument speichern wollen, müssen Sie das Dokument zuerst unter einem anderen Namen speichern ("Datei>Speichern unter") und danach das Kommando "Datei>Speichern als ODF" aufrufen.

Beseitigen von Problemen

Wenn Sie den ODF Eintrag im Microsoft Word Menü nicht finden, kann dies an einer fehlenden Word-Komponente (siehe auch Software-Voraussetzungen) oder an einer fehlerhaften Installation der "Word Primary Interop Assemblies" liegen. In einem solchen Fall können Sie das Add-In folgendermaßen händisch aktivieren.

Microsoft Word XP/2003

  1. Rufen Sie das Kommando "Anpassen" aus dem "Extras"-Menü auf und wählen Sie den "Befehle"-Karteireiter.
  2. In der "Kategorien"-Liste wählen Sie "Extras".
  3. Ziehen Sie den "COM Add-Ins" Eintrag aus der "Befehle"-Liste zu dem "Extras"-Menü. Wenn das "Extras"-Menü aufklappt, zeigen Sie auf die Stelle, an der der "COM Add-Ins"-Befehl eingefügt werden soll und lassen erst dann die Maustaste los.
  4. Klicken Sie auf "Schließen".
  5. Rufen Sie den "COM Add-Ins"-Befehl im "Extras"-Menü auf
  6. Aktivieren Sie das Kontrollkästchen für "ODF Add-In für Word 2003" im "COM Add-Ins" Dialog und klicken Sie auf "OK".

Microsoft Word 2007

  1. Rufen Sie "Word Optionen" im "Datei"-Menü auf
  2. Wählen Sie in der Combobox der "Add-Ins" Sektion den "COM Add-Ins"-Eintrag aus und klicken Sie auf "Go"
  3. Aktivieren Sie das Kontrollkästchen für "ODF Add-In für Word 2007" im "COM Add-Ins" Dialog und klicken Sie auf "OK".
