Complément OpenXML/ODF Translator pour Microsoft Word

Merci d'avoir téléchargé le complément OpenXML/ODF pour Word. Ce complément vous permet d'ouvrir, dans Microsoft Word, des documents basés sur les spécifications ODF et de démontrer l'interfonctionnement entre Office Open XML et ODF. Ce projet d'interfonctionnement est hébergé sur le site sourceforge.net. Pour plus de détails et pour des informations sur les futures mises à jour concernant le projet, veuillez consulter notre feuille de route (RoadMap).

Financé par Microsoft, le projet de traduction OpenXML/ODF Translator a été développé par Clever Age, Aztecsoft, Sonata et DIaLOGIKa (pour plus d'informations, veuillez visiter le site Web relatif au projet).

Configuration logicielle requise

Avant d'installer le complément, vérifiez que vous disposez de l'un des éléments suivants...

  • Microsoft Word 2000
  • Module de compatibilité Office
  • .NET Framework 2.0

ou

  • Microsoft Word XP
  • Module de compatibilité Office
  • .NET Framework 2.0

ou

  • Microsoft Word 2003
  • Module de compatibilité Office
  • .NET Framework 2.0*

ou

  • Microsoft Word 2007 (avec l'option Prise en charge de la programmabilité .NET activée)
  • .NET Framework 2.0*

(*) Remarque : Si vous avez installé Microsoft Word avant .NET Framework, vous devez mettre à jour votre installation de Word comme suit :

  1. Ouvrez Ajout/Suppression de programmes dans le Panneau de configuration.
  2. Recherchez l'entrée Microsoft Word et choisissez Modifier.
  3. Au cours du processus d'installation, choisissez Ajouter ou supprimer des composants puis ajoutez Prise en charge de la programmabilité .NET dans la section Microsoft Office Word.

Installation du complément

Effectuez un double clic sur le fichier « OdfAddInForWordSetup-en.exe »

Si l'installation réussit, vous pouvez ouvrir un fichier .ODT (ODT est l'extension de fichier standard des documents ODF) dans Microsoft Word (voir le menu Fichier>ODF...). En outre, lorsque vous effectuez un clic droit sur un fichier .ODT dans l'Explorateur, vous devez pouvoir ouvrir le document dans Microsoft Word (menu Ouvrir dans Word).

Remarque importante : Les fichiers ODT ouverts par le complément sont importés dans Microsoft Word en tant que fichiers DOCX en lecture seule. Par conséquent, si vous avez apporté des modifications à un fichier et que vous l'avez enregistré en tant que fichier .ODT, vous devez d'abord l'enregistrer sous un nouveau nom (Fichier>Enregistrer sous), puis utiliser la commande Fichier>ODF…-Enregistrer sous ODF….

Dépannage

Si l'entrée ODF n'apparaît pas dans le menu Fichier de Microsoft Word, il est possible qu'il manque des fonctionnalités dans Word (voir la section Configuration logicielle requise) ou qu'une panne se soit produite lors de l'installation des assemblys PIA (Primary Interop Assemblies) de Word.

Veuillez consulter notre Guide de dépannage pour plus d'informations.
