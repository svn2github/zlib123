OpenXML/ODF Translator Add-in for Microsoft Office

Merci d'avoir téléchargé OpenXML/ODF Translator Add-in for Microsoft Office. Le complément vous permet d'ouvrir des documents basés sur la spécification OpenDocument Format (ODF) dans Microsoft Office, et de démontrer ainsi l'interopérabilité entre Office Open XML et ODF. Ce projet d'interopérabilité est hébergé sur sourceforge.net. Pour plus d'informations et futures mises à jour sur le projet, consultez le calendrier de lancement.

Fondé par Microsoft, OpenXML/ODF Translator a été développé par Clever Age, Aztecsoft, Sonata et DIaLOGIKa (pour plus d'informations, reportez-vous au site Web du projet).

Configuration logicielle requise

Avant de procéder à l'installation du complément, assurez-vous que vous possédez au moins un des éléments suivants :

  • Microsoft Word 2000
  • Pack de compatibilité Microsoft Office
  • .NET Framework 2.0

ou

  • Microsoft Office XP
  • Pack de compatibilité Microsoft Office
  • .NET Framework 2.0

ou

  • Microsoft Office 2003
  • Pack de compatibilité Microsoft Office
  • .NET Framework 2.0

ou

  • Microsoft Office 2007
  • .NET Framework 2.0

Assurez-vous que toutes les mises à jours Windows et Office recommandées sont bien installées.

* Nous vous recommandons fortement d'installer le .NET Framework 2.0 Service Pack 1.

Installation du complément

Double-cliquez sur le fichier "OdfAddInForOfficeSetup-fr.exe" pour lancer l'installation et suivez le guide de configuration.

Si l'installation a réussi, vous devez pouvoir ouvrir les fichiers ODF sous Microsoft Word, Excel ou Powerpoint (reportez-vous au menu File => ODF...). Aussi, lorsque vous cliquez avec le bouton droit de la souris sur un fichier ODF dans Windows Explorer, vous devez avoir la possibilité d'ouvrir le document sous Microsoft Word, Excel ou PowerPoint (menu Ouvrir avec Word/Excel/PowerPoint).

REMARQUE IMPORTANTE : Les fichiers ODF ouverts avec le complément sont importés dans Microsoft Office comme documents en lecture seule. Par conséquent, si vous effectuez des modifications à un tel document et que vous souhaitez l'exporter en fichier ODF, vous devez d'abord l'enregistrer sous un autre nom (Fichier>Enregistrer sous...), puis utilisez la commande Fichier>ODF…-Enregistrer sous forme ODF….

Dépannage

Si l'installation échoue, reportez-vous à la section sur la Configuration logicielle requise. Assurez-vous que toutes les mises à jours Windows et Office recommandées sont bien installées.

Notez que vous devez redémarrer toutes les applications Office (par exemple, Word, Excel, PowerPoint) pour que les compléments soient chargés correctement. Vous devrez peut-être également redémarrer Outlook si vous travaillez sous Office XP ou 2003.

Si cela ne résout pas le problème, vous trouverez des instructions de dépannage détaillées sur le site Web du projet. Pour plus d'informations, consultez notre Guide d'installation.
