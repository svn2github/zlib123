Dodatek OpenXML/ODF Translator dla pakietu Microsoft Office

Dziękujemy za pobranie dodatku OpenXML/ODF Translator dla pakietu Microsoft Office. Dodatek umożliwia otwieranie dokumentów w standardzie OpenDocument Format (ODF) w programach pakietu Microsoft Office. Jest to przykład współdziałania między specyfikacją Office Open XML i ODF. Ten projekt współdziałania jest prowadzony w witrynie sourceforge.net. Więcej informacji oraz przyszłe aktualizacje dotyczące projektu można znaleźć w dokumencie planu (RoadMap).

Dodatek OpenXML/ODF Translator został opracowany na zlecenie firmy Microsoft przez firmy Clever Age, Aztecsoft, Sonata oraz DIaLOGIKa (więcej informacji można znaleźć w witrynie sieci Web projektu).

Instalacja dodatku

Aby rozpocząć instalację, należy kliknąć dwukrotnie plik „OdfAddInForOfficeSetup-pl.exe” i postępować zgodnie z instrukcjami przewodnika instalacji.

Po pomyślnym zainstalowaniu dodatku pliki w formacie ODF powinny otwierać się w programach Microsoft Word, Excel lub PowerPoint (patrz menu Plik => ODF...). Ponadto po kliknięciu prawym przyciskiem myszy pliku w formacie ODF w Eksploratorze Windows powinna zostać wyświetlona opcja otworzenia dokumentu w programie Microsoft Word, Excel lub PowerPoint (menu Otwórz za pomocą programu Word).

Ważna uwaga: Pliki ODF otwarte z wykorzystaniem dodatku są importowane do pakietu Microsoft Office jako dokumenty tylko do odczytu. Jeżeli więc wprowadzono zmiany to, aby wyeksportować dokument z powrotem jako plik w formacie ODF, należy najpierw zapisać go z nową nazwą (Plik>Zapisz jako), a następnie użyć polecenia Plik>ODF...-Zapisz jako ODF....

Rozwiązywanie problemów

Jeżeli instalacja nie powiedzie się, należy ponownie zapoznać się z sekcją dotyczącą wymagań programowych. Należy sprawdzić, czy zainstalowane są wszystkie zalecane aktualizacje systemu Windows i pakietu Office.

Należy pamiętać, że w celu uruchomienia dodatku należy najpierw ponownie uruchomić wszystkie aplikacje pakietu Office (tj. programy Word, Excel, PowerPoint). W przypadku pakietu Office XP lub 2003 konieczne może być również ponowne uruchomienie programu Outlook.

Jeżeli w dalszym ciągu występują problemy, szczegółowe instrukcje rozwiązywania problemów można znaleźć w witrynie sieci Web projektu. Więcej informacji można znaleźć w przewodniku instalacji.
