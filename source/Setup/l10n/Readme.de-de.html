OpenXML/ODF Translator Add-In für Microsoft Office

Herzlichen Dank, dass Sie das OpenXML/ODF Translator Add-In für Microsoft Office heruntergeladen haben. Mit diesem Add-In können Sie Dokumente, die auf der Open Document Format (ODF) Spezifikation basieren, in Microsoft Office öffnen bzw. mit Office erstellte Dokumente in diesem Format speichern. Damit ist die Interoperabilität zwischen den Spezifikationen Office OpenXML (ECMA) und ODF gewährleistet. Das Projekt OpenXML/ODF Translator Add-In für Microsoft Office wird auf sourceforge.net. gehosted. Weitere Einzelheiten und zukünftige Updates entnehmen Sie bitte dort dem Dokument "RoadMap".

Der OpenXML/ODF Translator wird von Microsoft finanziell unterstützt und von Clever Age, Aztecsoft, Sonata und DIaLOGIKa entwickelt (mehr Informationen darüber finden Sie hier).

Das Add-In installieren

Machen Sie einen Doppelklick auf "OdfAddInForOfficeSetup-de.exe" und folgen Sie den Installationsanweisungen.

Nach erfolgreicher Installation können Sie .ODT Dokumente (das ist die Dateierweiterung für ODF Text Dokumente) in Microsoft Word öffnen (siehe Kommando Datei>ODF). Auch können Sie .ODT Dokumente im Explorer durch Klick auf die rechte Maustaste mit Microsoft Word öffnen (Kommando mit Word/Excel/PowerPoint öffnen).

Wichtige Anmerkung: Das .ODT Dokument wird als schreibgeschütztes .DOCX Dokument in Microsoft Word geöffnet. Wenn Sie in das Dokument verändert haben und es wieder als .ODT Dokument speichern wollen, müssen Sie das Dokument zuerst unter einem anderen Namen speichern (Datei>Speichern unter) und danach das Kommando Datei>Speichern als ODF aufrufen.

Beseitigen von Problemen

Sollte die Installation fehlschlagen prüfen Sie bitte die Software-Voraussetzungen. Stellen Sie sicher, dass alle empfohlenen Windows- und Office-Updates installiert sind.

Bitte beachten Sie, dass die Add-Ins erst nach einem Neustart der Office-Anwendungen (Word, Excel, PowerPoint) geladen werden. Bei Office XP und 2003 kann es zusätzlich notwendig sein, Outlook neu zu starten.

Weitere Informationen zur Installation und Hilfestellung bei Problemen finden Sie auch in unserem Troubleshooting Guide.
