Microsoft Word OpenXML/ODF 转换器加载项

感谢您下载 Word ODF OpenXML/ODF 转换器加载项。本加载项使您能够在 Microsoft Word 中打开基于 ODF 规格的文档，从而展现 Office Open XML 和 ODF 之间的互操作性。此互操作性项目寄存在 sourceforge.net。如需有关此项目的更多细节和未来更新，请查看“路线图（RoadMap）”文档。

由 Microsoft 出资的 OpenXML/ODF 转换器已由 Clever Age、Aztecsoft、Sonata 和 DIaLOGIKa 开发（如需更多信息，请参见项目网站）。

软件要求

在安装加载项之前，确保您拥有下列项目之一...

  • Microsoft Word 2000
  • Office 兼容程序包
  • .NET Framework 2.0

或者

  • Microsoft Word XP
  • Office 兼容程序包
  • .NET Framework 2.0

或者

  • Microsoft Word 2003
  • Office 兼容程序包
  • .NET Framework 2.0*

或者

  • Microsoft Word 2007 （已激活 .NET 可编程性支持）
  • .NET Framework 2.0*

（*）注：如果您先安装 Microsoft Word，再安装 .NET Framework，您将需要通过运行以下步骤，更新您的 Word 安装。

  1. 从控制面板打开“添加或删除程序”
  2. 找到 Microsoft Word 条目并选择“修改”
  3. 在安装进程中，选择“添加或删功能”，然后在“Microsoft Office Word”部分中添加“.NET 可编程支持”。

安装加载项

双击“OdfAddInForWordSetup-chs.exe”文件

如果安装成功，您应该能够用 Microsoft Word （参见文件>ODF...菜单）打开 .ODT 文件（ODT 是 ODF 文本文档的标准文件扩展名）同时，当在资源管理器中右击 .ODT 文件时，您应能看到在 Microsoft Word 中打开该文档的选项（用 Word 打开菜单）。

重要注释：由该加载项打开的 .ODT 文件，可在 Microsoft Word 中作为只读 DOCX 文件导入。因此，如果您更改该文件并希望将其保存回 .ODT 文件，则您首先需要用一个新的名称保存该文件（文件>另存为），然后使用文件>ODF…-另存为 ODF… 命令。

故障排除

如果您未在 Microsoft Word 文件菜单中看到 ODF 项，这可归因于 Word 缺少功能（参见软件要求部分）或者未能安装 Word 主互操作程序集。
