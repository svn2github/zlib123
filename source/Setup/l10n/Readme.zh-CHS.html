适用于 Microsoft Office 的 OpenXML/ODF 变换器加载项

感谢您下载适用于 Microsoft Office 的 OpenXML/ODF 变换器加载项。该加载项能够使您在 Microsoft Office 中的 OpenDocument 格式 (ODF) 规格的基础上打开文档，因此，也展示了 Office Open XML 和 ODF 之间的互操作性。在 sourceforge.net 上将提供该互操作性项目。有关该项目的详细信息以及未来更新，请参阅 RoadMap 文档。

OpenXML/ODF 变换器由 Microsoft 出资，由 Clever Age、Aztecsoft、Sonata 和 DIaLOGIKa 负责开发（有关详细信息，请参阅该项目网站）。

软件要求

安装加载项之前，请确保具备以下应用程序之一：

  • Microsoft Word 2000
  • Office 兼容性工具包
  • .NET Framework 2.0

或

  • Microsoft Office XP
  • Office 兼容性工具包
  • .NET Framework 2.0

或

  • Microsoft Office 2003
  • Office 兼容性工具包
  • .NET Framework 2.0

或

  • Microsoft Office 2007
  • .NET Framework 2.0

请确保已安装所有建议的 Windows 和 Office 更新。

* 强烈建议安装 .NET Framework 2.0 Service Pack 1。

安装加载项

双击“OdfAddInForOfficeSetup-chs.exe”文件开始安装并遵循安装指南。

如果安装成功，您应当能够使用 Microsoft Word、Excel 或 PowerPoint 打开 ODF 文件（请参阅“文件”=>“ODF...”菜单）。同样，在 Windows 资源管理器中于 ODF 文件上右击时，您应当能够选择在 Microsoft Word、Excel 或 PowerPoint 中打开文档（请参阅“用 Word 打开”菜单）。

重要注意事项：通过加载项打开的 ODF 文件作为只读文档导入 Microsoft Office 中。因此，如果您对此类文档做出了更改并要将其作为 ODF 文件导回，您必须首先将其用一个新名称保存（“文件”>“另存为”），然后使用“文件”>“ODF…”-“另存为 ODF…” 命令。

故障排除

如果安装不成功，请再次查看软件要求上的相关部分。请确保已安装所有建议的 Windows 和 Office 更新。

请注意：对于要加载的加载项，必须重新启动所有 Office 应用程序（例如，Word、Excel、PowerPoint）。使用 Office XP 或 2003 时，可能也要重新启动 Outlook。

如果这些都不能帮助您解决问题，请您在该项目网站上查找详细的故障排除说明。有关更多信息，请查看安装指南。
