OpenXML/ODF Translator Add-in for Microsoft Office

Thank you for downloading the OpenXML/ODF Translator Add-in for Microsoft Office. The add-in enables you to open documents based on the OpenDocument Format (ODF) specification in Microsoft Office, thus, demonstrating the interoperability between Office Open XML and ODF. This interoperability project is hosted on sourceforge.net. For more details and future updates on the project, please see the RoadMap document.

Funded by Microsoft the OpenXML/ODF Translator has been developed by Clever Age, Aztecsoft, Sonata and DIaLOGIKa (for more information, see the project web site).

Software Requirements

Before installing the add-in, make sure you have one of the following:

  • Microsoft Word 2000
  • Office Compatibility Pack
  • .NET Framework 2.0

or

  • Microsoft Office XP
  • Office Compatibility Pack
  • .NET Framework 2.0

or

  • Microsoft Office 2003
  • Office Compatibility Pack
  • .NET Framework 2.0

or

  • Microsoft Office 2007
  • .NET Framework 2.0

Please make sure all recommended Windows and Office updates are installed.

* The installation of .NET Framework 2.0 Service Pack 1 is highly recommended.

Installing the Add-in

Double click the "OdfAddInForOfficeSetup-en.exe" file to start the installation and follow the setup guide.

If the installation is successful, you should be able to open an ODF files using Microsoft Word, Excel or PowerPoint (see File => ODF... menu). Also, when right-clicking on an ODF file in Windows Explorer, you should have the option of opening the document in Microsoft Word, Excel or PowerPoint (Open with Word/Excel/PowerPoint menu).

Important note: ODF files opened by the add-in are imported in Microsoft Office as read-only documents. Consequently, if you have made changes to such a document and want to export it back as an ODF file, you will have first to save it with a new name (File>Save As) and then use the File>ODF…-Save as ODF… command.

Troubleshooting

If the installation does not succeed please check the section on Software Requirements again. Make sure all recommended Windows and Office updates are installed.

Please note that for the add-ins to be loaded all Office applications (i.e. Word, Excel, PowerPoint) must be restarted. When using Office XP or 2003 it might be necessary to restart Outlook as well.

If this does not help you will find detailed troubleshooting instructions on the project website. Please check our install guide for further information.
