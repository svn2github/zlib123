ODF Add-in pour Microsoft Word

Merci d'avoir télécharger l'add-in ODF pour Microsoft Word. Le but de cet add-in est de permettre aux utilisateurs d'ouvrir des documents au format ODF directement depuis Microsoft Word, demontrant ainsi l'inter-operabilité entre les formats OpenXML et ODF. Le projet est hébergé sur sourceforge.net. Pour plus d'information sur le projet ainsi que pour les prochaines releases, merci de bien vouloir consulter notre roadmap.

Pre-Requis

Avant d'installer le produit, veuillez vous assurer de bien respecter la liste des pre-requis suivante....

  • Microsoft Word XP
  • Office Compatibility Pack
  • .NET framework 2.0

ou

  • Microsoft Word 2003
  • Office Compatibility Pack
  • .NET framework 2.0*

ou

  • Word 2007 avec activation du support programmable .NET
  • .NET framework 2.0*

(*) Attention: si vous avez installer Microsoft Word avant le framework .NET, il vous faudra mettre à jour votre installation de Microsoft Word en suivant les instructions suivantes:

  1. Dans le panneau de configuration, ouvrir "Ajout/Suppression de programmes"
  2. Dans la liste, selectionez "Microsoft Office" et cliquez sur "Modifier"
  3. Dans la boite de dialoque sélectionnez "Ajouter ou supprimer des fonctionnalitées".
  4. Sélectionnez ensuite ".NET Programmability Support" dans la section "Microsoft Office Word".

Installation de l'add-in

Double clickez sur le fichier "OdfAddInForWordSetup-fr-1.0.exe"

Si l'installation s'est correctement déroulée vous devriez pouvoir ouvrire un fichier ".ODT" depuis Microsoft Word (voir menu "Fichier>ODF...>Ouvrir un fichier au format OpenDocument (ODF)"). Aussi, l'explorateur Windows devrait afficher l'option "Ouvrir avec Micosoft Word" dans son menu contextuel.

Important:Les fichiers au format ODF sont ouverts dans Microsoft Word en lecture seule et au format OpenXML (".DOCX"). Si vous voulez modifier le contenu du fichier il vous faudra au préalable faire une copie du document (menu "Fichier>Enregister Sous"). La sauvegarde au format ODF s'effectue en sélectionnant le menu "Fichier>ODF...>Enregistrer au format ODF...".

Résolution des problèmes

Si le menu "ODF..." n'apparait pas sous Word, il est possible que certains pre-requis ne soient pas vérifié (voir la section Pre-Requis) ou bien que l'installation de certains composants est échouée. Il vous faudra alors activer l'add-in manuellement dans Word:

Microsoft Word XP/2003

  1. Sélectionnez le menu "Outils>Personnaliser...".
  2. Sous l'onglet "Commandes", séléctionnez "Outils" dans la liste "Catégories".
  3. Glissez-déposez le composant "Modules complémentaires COM..." depuis la liste "Commandes" vers le menu "Outils".
  4. Cliquez sur "Fermer".
  5. Dans le menu "Outils", sélectionnez "Modules complémentaires COM..."
  6. Sélectionnez ensuite le module "ODF Add-In for Word XP".

Microsoft Word 2007

  1. Cliquez "Options pour Word" dans le menu"Fichier"
  2. Dans la section "Add-ins", sélectionnez dans la liste "Modules complémentaires COM...". Cliquez ensuite sur le bouton "Aller".
  3. Sélectionnez ensuite le module "ODF Word 2007 Add-in" et cliquez sur "OK".
