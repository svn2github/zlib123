ODF Add-In für Excel XP

Vielen Dank für das Herunterladen des ODF Add-Ins für Excel XP. Die Software ermöglicht das Öffnen eines auf der ODF-Spezifikation basierenden Dokuments in Excel XP und demonstriert die Interoperabilität zwischen den OpenXML- und ODF-Spezifikationen.

Softwareanforderungen

Zum Testen des Add-Ins ist Folgendes erforderlich:
1. Excel XP
2. .NET Framework 2.0*
3. Compatibility Pack* für Word-, Excel- und PowerPoint 2007-Dateiformate (mindestens Beta 2).

Wichtige Hinweise:

(*) Stellen Sie sicher, dass das vollständige Office-Update ausgeführt wird, bevor Sie das Compatibility Pack installieren. Das Office-Update ist über Office Online (http://office.microsoft.com/en-us/default.aspx) verfügbar.


Installieren des Add-Ins

Führen Sie die heruntergeladene Datei setup.exe aus und folgen Sie den Anweisungen.

Nachdem die Installation erfolgreich abgeschlossen wurde, wird im Excel-Menü "Datei" die Option "ODF öffnen" angezeigt. Sie können die ODF-Arbeitsblattdatei importieren oder das aktuelle Dokument als ODF-Arbeitsblattdatei exportieren (während der Entwicklung sind diese Funktionen möglicherweise vorübergehend nicht verfügbar).

Wichtiger Hinweis: Die mit dem Add-In geöffnete ODF-Datei wird als schreibgeschützte XLSX-Datei in Microsoft Excel importiert. Um die Datei wieder als ODF-Datei zu speichern, müssen Sie zuerst eine Kopie des Dokuments ("Menü > Speichern unter") erstellen und anschließend die Menüoption “Als ODF exportieren" auswählen.

Problembehandlung

Wenn die ODF-Option nicht angezeigt wird, sind möglicherweise Features in Excel nicht vorhanden (siehe Softwareanforderungen). Nachdem die Installation des ODF Add-ins abgeschlossen ist, gehen Sie wie folgt vor, um das Add-In zu aktivieren:
* Wählen Sie "Tools > Anpassen" aus und klicken Sie auf die Registerkarte "Befehle".
* Klicken Sie im Feld "Kategorien" auf "Tools".
* Ziehen Sie "COM Add-Ins" aus dem Feld "Befehle" in das Menü "Tools". Wenn die Befehle im Menü "Tools" angezeigt werden, zeigen Sie auf die Stelle, an der die COM Add-Ins eingefügt werden sollen, und lassen Sie die Maustaste los.
* Klicken Sie auf "Schließen".
* Klicken Sie im Menü "Tools" auf "COM Add-Ins" und aktivieren Sie in der Liste das Kontrollkästchen "ODF Add-In". Wenn das gewünschte Add-In nicht in der Liste aufgeführt ist, klicken Sie auf "Hinzufügen", wählen Sie das Add-In aus und klicken Sie auf "OK".
