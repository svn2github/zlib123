ODF Add-in for Excel XP (Excel XP 用 ODF アドイン プログラム)

ODF Add-in for Excel XP をダウンロードいただき、ありがとうございます。このソフトウェアをご使用になると、ODF 仕様のドキュメントを Excel XP で開くことができ、OpenXML と ODF 仕様との間の相互運用性を発揮することができまこ

ソフトウェアの要件

このアドイン プログラムをテストするには、以下の要件を満たしていただく必要があります。
1. Excel XP
2. .NET Framework 2.0* & Shared add-in extensibility(KB908002)***
3. Word、Excel、および PowerPoint 2007 ファイル形式と互換性のある互換機能パック* (ベータ 2 またはそれ以上のバージョン)。

重要事項:

(**) 必ず最新の Office 更新プログラムを実行してから、互換機能パックをインストールしてください。Office の更新プログラムをダウンロードするには、Office Online (http://office.microsoft.com/en-us/default.aspx) にアクセスしてください。

(***) Microsoft 共有アドイン サポート更新プログラム (KB908002) は、アドインのセットアップ中にインストールされます。

アドインのインストール

以下の手順に従い、ダウンロードした setup.exe プログラムをインストールしてください。

インストールが完了すると、Excel の [ファイル] メニューに [ODF を開く] が追加され、ODF スプレッドシート ファイルをインポートするか、または現在作業中のドキュメントを ODF スプレッドシート ファイルにエクスポートすることができます (注意: 開発段階で、これらの機能が一時的に使用できない場合がありますので、ご了承ください)。

重要事項:アドインが開いた ODF ファイルは、Microsoft Excel の読み取り専用の XLSX ファイル形式です。ODF ファイル形式で保存し直す場合は、[ファイル] メニューの [名前を付けて保存] を選択して、そのドキュメントをいったん保存してから、[ODF 形式で保存] を選択します。

トラブルシューティング ガイド

メニューに ODFが表示されない場合は、Excel の必須機能がインストールされていない可能性があります (上記の「ソフトウェアの要件」をご覧ください)、または Excel プライマリ相互運用機能アセンブリが正常にインストールされなかった可能性があります。

詳細については、トラブルシューティング ガイド を参照してください。
