Complément ODF pour Excel XP

Merci d’avoir téléchargé le complément ODF pour Excel XP. Grâce à ce logiciel, les utilisateurs peuvent ouvrir dans Excel XP des documents conformes aux spécifications ODF et démontrer l'interfonctionnement entre les spécifications OpenXML et ODF.

Configuration logicielle requise

Vous devrez disposer des éléments suivants pour tester le complément :
1. Excel XP
2. .NET Framework 2.0
3. Module de compatilibité* pour les formats de fichier Word, Excel et PowerPoint 2007 (version bêta 2 ou ultérieure)

REMARQUES IMPORTANTES :

(*) Prenez soin d'exécuter entièrement la mise à jour d'Office avant d'installer le module de compatibilité. La mise à jour d'Office est disponible sur le site Office Online (http://office.microsoft.com/en-us/default.aspx).


Installation du complément

Exécutez le programme setup.exe téléchargé et effectuez les étapes suivantes...

Si l'installation réussit, une nouvelle entrée « Open ODF » apparaît dans le menu « Fichier » d'Excel. Elle permet d'importer une feuille de calcul ODF ou d'exporter votre document de travail en cours en tant que feuille de calcul ODF (notez que ces fonctionnalités risquent d'être temporairement indisponibles lors du processus de développement).

Remarque importante : Les fichiers ODF ouverts par le complément sont importés dans Microsoft Excel en tant que fichiers XLSX en lecture seule. Pour enregistrer à nouveau ce document en tant que fichier ODF, effectuez-en d'abord une copie (« Menu>Enregistrer sous »), puis utilisez « Enregistrer sous ODF ».

Guide de dépannage

Si l'entrée ODF n'apparaît pas, il est possible qu'il manque des fonctionnalités dans Excel (voir la rubrique Configuration logicielle requise, plus haut). Après l'installation du Complément ODF, vous devrez activer ce dernier en effectuant les actions suivantes :
* Allez à « Outils > Personnaliser », puis cliquez sur l'onglet « Commandes ».
* Dans la zone « Catégories », cliquez sur « Outils ».
* Faites glisser la commande Modules complémentaires COM de la zone « Commandes » vers le menu « Outils ». Lorsque le menu « Outils » affiche les commandes de menu, pointez vers l'emplacement du menu où vous souhaitez insérer la commande Modules complémentaires COM, puis relâchez le bouton de la souris.
* Cliquez sur Fermer.
* Dans le menu Outils, cliquez sur « Modules complémentaires COM », puis activez la case à cocher adjacente au nom du complément dans la liste Macros complémentaires disponibles. Si le complément souhaité ne se trouve pas dans la liste, cliquez sur Ajouter, recherchez le complément, puis cliquez sur OK.
