Excel XP ODF 加载项

感谢您下载 Excel XP ODF 加载项。该软件的目的在于使用户能够用 Microsoft Excel 打开基于 ODF 规格的文档，并展示 OpenXML 和 ODF 规格之间的互操作性。

软件要求

若要测试此加载项，您将需要以下工具：
1. Excel XP
2. .NET Framework 2.0
3. Word、Excel 和 PowerPoint 2007 文件格式兼容程序包*（beta 2 或更高版本）。

重要注释：

（*）请确保在安装兼容程序包之前，完整运行 Office 更新。Office 更新可从 Office Online（http://office.microsoft.com/en-us/default.aspx）获取


安装加载项

运行已下载的 setup.exe 程序并按步进行...

如果安装成功，您应该在 Excel 的“文件”菜单中看到新的“Open ODF”项。该菜单项允许导入 ODF 电子表格文件，或将当前编辑的文档导出为 ODF 电子表格文件（注意：在开发进程中，这些功能可能临时不可用）。

重要注释：由该加载项打开的 ODF 文件，可在 Microsoft Excel 中作为只读 XLSX 文件导入。如果您想要将其再次保存为 ODF，那么您必须先为该文档做一副本（“菜单>另存为”），然后用“导出为 ODF”菜单。

故障排除指南

如果未看到 ODF 项，则可能因为 Excel 中缺少某些功能（查看上文列出的软件要求）。在完成 ODF 加载项安装之后，您需要通过以下操作激活该加载项：
* 转到“工具 > 自定义”，然后单击“命令”选项卡。
* 在“类别”框中，单击“工具”。
* 将 COM 加载项从“命令”框拖至“工具”菜单。当“工具”菜单显示菜单命令时，指向您希望显示 COM 加载项命令的菜单位置，然后松开鼠标按钮。
* 单击“关闭”。
* 在“工具”菜单中，单击“COM 加载项”，并选定可用列表中 ODF 加载项旁的复选框。如果您想要的加载项未在列表上，单击“添加”，找到该加载项，然后单击“确定”。
