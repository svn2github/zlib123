ODF Add-In für Excel 2007

Vielen Dank für das Herunterladen des ODF Add-Ins für Excel 2007. Die Software ermöglicht das Öffnen eines auf der ODF-Spezifikation basierenden Dokuments in Excel 2007 und demonstriert die Interoperabilität zwischen den OpenXML- und ODF-Spezifikationen.

Softwareanforderungen

Zum Testen des Add-Ins ist Folgendes erforderlich:
1. Excel 2007
2. .NET Framework 2.0*

Wichtige Hinweise:

(*) Warnung: Wenn Sie Microsoft Excel bevor .NET Framework installieren, müssen Sie die Excel-Installation aktualisieren, indem Sie folgende Schritte ausführen.

1. Doppelklicken Sie in der Systemsteuerung auf die Option "Software".
2. Wählen Sie Microsoft Excel aus und klicken Sie auf "Ändern".
3. Wählen Sie "Features hinzufügen/entfernen" aus und fügen Sie ".Net-Programmierunterstützung" im Abschnitt "Microsoft Office Excel" hinzu.

Installieren des Add-Ins

Führen Sie die heruntergeladene Installationsprogramm aus und folgen Sie den Anweisungen.

Nachdem die Installation erfolgreich abgeschlossen wurde, wird im Excel-Menü "Datei" die Option "ODF öffnen" angezeigt. Sie können die ODF-Arbeitsblattdatei importieren oder das aktuelle Dokument als ODF-Arbeitsblattdatei exportieren (während der Entwicklung sind diese Funktionen möglicherweise vorübergehend nicht verfügbar).

Wichtiger Hinweis: Die mit dem Add-In geöffnete ODF-Datei wird als schreibgeschützte DOCX-Datei in Microsoft Excel importiert. Um die Datei wieder als ODF-Datei zu speichern, müssen Sie zuerst eine Kopie des Dokuments ("Menü > Speichern unter") erstellen und anschließend die Menüoption “Als ODF exportieren" auswählen.

Problembehandlung

Wenn die ODF-Option nicht angezeigt wird, sind möglicherweise Features in Excel nicht vorhanden (siehe Softwareanforderungen). Nachdem die Installation des ODF Add-ins abgeschlossen ist, gehen Sie wie folgt vor, um das Add-In zu aktivieren:

* Klicken Sie im Menü "Datei" auf "Excel-Optionen".
* Wählen Sie unter "Add-Ins" im Listenfeld die Option "COM Add-Ins" aus und klicken Sie auf "Weiter".
* Aktivieren Sie das Kontrollkästchen "ODF Excel 2007 Add-In" und klicken Sie auf "OK".
