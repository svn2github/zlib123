Complément ODF pour Excel 2007

Merci d'avoir téléchargé le complément ODF pour Excel 2007. Grâce à ce logiciel, les utilisateurs peuvent ouvrir dans Excel 2007 des documents conformes aux spécifications ODF et démontrer l'interfonctionnement entre les spécifications OpenXML et ODF.

Configuration logicielle requise

Vous devrez disposer des éléments suivants pour tester le complément :
1. Excel 2007
2. .NET Framework 2.0*

REMARQUES IMPORTANTES :

(*) Avertissement : Si vous installez Microsoft Excel avant .NET Framework, vous devrez mettre à jour votre installation d'Excel, en effectuant les étapes suivantes :

1. Ouvrez « Ajout/Suppression de programmes dans le Panneau de configuration.
2. Recherchez Microsoft Excel et choisissez « Modifier ».
3. Au cours du processus d'installation, choisissez « Ajouter/supprimer des composants » et ajoutez « Prise en charge de la programmabilité .NET dans la section « Microsoft Office Excel ».

Installation du complément

Exécutez le programme d'installation téléchargé et effectuez les étapes suivantes...

Si l'installation réussit, une nouvelle entrée « Open ODF » apparaît dans le menu « Fichier » d'Excel. Elle permet d'importer une feuille de calcul ODF ou d'exporter votre document de travail en cours en tant que feuille de calcul ODF (notez que ces fonctionnalités risquent d'être temporairement indisponibles lors du processus de développement).

Remarque importante : Les fichiers ODF ouverts par le complément sont importés dans Microsoft Excel en tant que fichiers DOCX en lecture seule. Pour enregistrer à nouveau ce document en tant que fichier ODF, effectuez-en d'abord une copie (« Menu>Enregistrer sous »), puis utilisez « Enregistrer sous ODF ».

Guide de dépannage

Si l'entrée ODF n'apparaît pas, il est possible qu'il manque des fonctionnalités dans Excel (voir la rubrique Configuration logicielle requise, plus haut). Après l'installation du Complément ODF, vous devrez activer ce dernier en effectuant les actions suivantes :

* Dans le menu Fichier, cliquez sur « Options Excel ».
* Dans la section Compléments, sélectionnez Compléments COM dans la zone de liste, puis cliquez sur le bouton OK.
* Prenez soin d'activer la case à cocher « Complément ODF pour Excel 2007 » et de valider votre choix en cliquant sur OK.
