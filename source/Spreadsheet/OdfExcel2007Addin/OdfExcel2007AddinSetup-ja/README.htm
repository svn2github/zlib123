ODF Add-in for Excel 2007 (Excel 2007 用 ODF アドイン プログラム)

ODF Add-in for Excel 2007 をダウンロードいただき、ありがとうございます。このソフトウェアをご使用になると、ODF 仕様のドキュメントを Excel 2007 で開くことができ、OpenXML と ODF 仕様との間の相互運用性を発揮することができまこ

ソフトウェアの要件

このアドイン プログラムをテストするには、以下の要件を満たしていただく必要があります。
1. Excel 2007
2. .NET Framework 2.0*

重要事項:

(*)ご注意:Microsoft Excel が既にインストールされている状態で .Net Framework をインストールした場合は、次の手順に従って Excel を更新してください。

1. [コントロール パネル] の [ソフトウェアの追加と削除] をクリックします。
2. Microsoft Excel の [変更] をクリックします。
3. インストール プログラムの [機能の追加と削除] を選択し、[Microsoft Office Excel の .NET プログラミング サポート] を追加します。

アドインのインストール

以下の手順に従い、ダウンロードしたインストール プログラムをインストールしてください。

インストールが完了すると、Excel の [ファイル] メニューに [ODF を開く] が追加され、ODF スプレッドシート ファイルをインポートするか、または現在作業中のドキュメントを ODF スプレッドシート ファイルにエクスポートすることができます (注意: 開発段階で、これらの機能が一時的に使用できない場合がありますので、ご了承ください)。

重要事項:アドインが開いた ODF ファイルは、Microsoft Excel の読み取り専用の DOCX ファイル形式です。ODF ファイル形式で保存し直す場合は、[ファイル] メニューの [名前を付けて保存] を選択して、そのドキュメントをいったん保存してから、[ODF 形式で保存] を選択します。

トラブルシューティング ガイド

メニューに ODFが表示されない場合は、Excel の必須機能がインストールされていない可能性があります (上記の「ソフトウェアの要件」をご覧ください)、または Excel プライマリ相互運用機能アセンブリが正常にインストールされなかった可能性があります。

詳細については、トラブルシューティング ガイド を参照してください。
