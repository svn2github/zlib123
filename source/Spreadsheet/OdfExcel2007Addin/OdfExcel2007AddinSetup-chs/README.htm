Excel 2007 ODF 加载项

感谢您下载 Excel 2007 ODF 加载项。该软件之目的在于使用户能够在 Excel 2007 中打开基于 ODF 规格的文档，并展示 OpenXML 和 ODF 规格之间的互操作性。虽然此加载项仍是测试版，因此您可在自担风险的前提下，使用和测试它！而且，我们欢迎各类反馈！ ;)

软件要求

若要测试此加载项，您将需要以下工具：
1. Excel 2007
2. .NET Framework 2.0*

重要注释：

（*）警告：如果您先安装 Microsoft Excel，再安装 .NET Framework，您将需要通过运行以下步骤，更新您的 Excel 安装。

1. 从控制面板打开“添加或删除程序”
2. 找到 Microsoft Excel 项并选择“修改”
3. 在安装进程中，选择“添加或删功能”，然后在“Microsoft Office Excel”部分中添加“.NET 可编程支持”。

安装加载项

运行已下载的安装程序并按步进行...

如果安装成功，您应该在 Excel 的“文件”菜单中看到新的“Open ODF”项。该菜单项允许导入 ODF 电子表格文件，或将当前编辑的文档导出为 ODF 电子表格文件（注意：在开发进程中，这些功能可能临时不可用）。

重要注释：由该加载项打开的 ODF 文件，可在 Microsoft Excel 中作为只读 DOCX 文件导入。如果您想要将其再次保存为 ODF，那么您必须先为该文档做一副本（“菜单>另存为”），然后用“导出为 ODF”菜单。

故障排除指南

如果未看到 ODF 项，则可能因为 Excel 中缺少某些功能（查看上文列出的软件要求）。在完成 ODF 加载项安装之后，您需要通过以下操作激活该加载项：

* 在“文件”菜单中单击“Excel 选项”
* 在“加载项”部分中，从列表框中选择“COM 加载项”，按“转到”按钮。
* 确保选择“ODF Excel 2007 加载项”复选框，并用“确定”按钮验证。
