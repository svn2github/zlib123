Complément OpenXML/ODF Translator pour Microsoft Excel

Merci d'avoir téléchargé le complément OpenXML/ODF pour Excel. Ce complément vous permet d'ouvrir, dans Microsoft Excel, des documents basés sur les spécifications ODF et de démontrer l'interfonctionnement entre Office Open XML et ODF. Ce projet d'interfonctionnement est hébergé sur le site sourceforge.net. Pour plus de détails et pour des informations sur les futures mises à jour concernant le projet, veuillez consulter notre feuille de route (RoadMap).

Financé par Microsoft, le projet de traduction OpenXML/ODF Translator a été développé par Clever Age et Sonata Software Limited (pour plus d'informations, veuillez visiter le site Web relatif au projet).

Configuration logicielle requise

Avant d'installer le complément, vérifiez que vous disposez de l'un des éléments suivants...

  • Microsoft Excel XP
  • Module de compatibilité Office
  • .NET Framework 2.0 SP1

ou

  • Microsoft Excel 2003
  • Module de compatibilité Office
  • .NET Framework 2.0* SP1

ou

  • Microsoft Excel 2007 (avec l'option Prise en charge de la programmabilité .NET activée)
  • .NET Framework 2.0* SP1

(*) Remarque : Si vous avez installé Microsoft Excel avant .NET Framework, vous devez mettre à jour votre installation de Excel comme suit :

  1. Ouvrez Ajout/Suppression de programmes dans le Panneau de configuration.
  2. Recherchez l'entrée Microsoft Excel et choisissez Modifier.
  3. Au cours du processus d'installation, choisissez Ajouter ou supprimer des composants puis ajoutez Prise en charge de la programmabilité .NET dans la section Microsoft Office Excel.

Installation du complément

Effectuez un double clic sur le fichier « OdfAddInForExcelSetup-en.exe »

Si l'installation réussit, vous pouvez ouvrir un fichier .ODS (ODS est l'extension de fichier standard des documents ODF) dans Microsoft Excel (voir le menu Fichier>ODF...). En outre, lorsque vous effectuez un clic droit sur un fichier .ODS dans l'Explorateur, vous devez pouvoir ouvrir le document dans Microsoft Excel (menu Ouvrir dans Excel).

Remarque importante : Les fichiers ODS ouverts par le complément sont importés dans Microsoft Excel en tant que fichiers XLSX en lecture seule. Par conséquent, si vous avez apporté des modifications à un fichier et que vous l'avez enregistré en tant que fichier .ODS, vous devez d'abord l'enregistrer sous un nouveau nom (Fichier>Enregistrer sous), puis utiliser la commande Fichier>ODF…-Enregistrer sous ODF….

Dépannage

Si l'entrée ODF n'apparaît pas dans le menu Fichier de Microsoft Excel, il est possible qu'il manque des fonctionnalités dans Excel(voir la section Configuration logicielle requise) ou qu'une panne se soit produite lors de l'installation des assemblys PIA (Primary Interop Assemblies) de Excel.

Veuillez consulter notre Guide de dépannage pour plus d'informations.
