Microsoft Word 用 ODF アドイン

Word 用 ODF アドインをダウンロードいただき、ありがとうございます。このソフトウェアの目的は、ODF 仕様のドキュメントを Microsoft Word で開けるようにし、オープン XML 仕様と ODF 仕様の相互運用を実現することです。この相互運用プロジェクトは、 sourceforge.net で運営されています。プロジェクトの詳細および将来の更新については、ロードマップのドキュメントを参照してください。

このアドインは、マイクロソフトが資金を支出し、Clever Age、DIaLOGIKa、および Aztecsoft が開発しました (詳細については、プロジェクトの Web サイトを参照してください)。

ソフトウェア要件

アドインをインストールする前に、次のいずれかがインストールされていることを確認してください。

  • Microsoft Word XP
  • Office 互換機能パック
  • .NET Framework 2.0

または

  • Microsoft Word 2003
  • Office 互換機能パック
  • .NET Framework 2.0*

または

  • Word 2007 (.NET プログラミング サポートを有効化)
  • .NET Framework 2.0*

(*) 警告 : .NET Framework をインストールする前に Microsoft Word をインストールした場合、次の手順に従って Word のインストール環境を更新する必要があります。

  1. コントロール パネルの [プログラムの追加と削除] を開きます。
  2. Microsoft Word を見つけて、[変更] をクリックします。
  3. インストール処理中に、[機能の追加または削除] を選択して、[Microsoft Office Word] セクションに [.NET プログラミング サポート] を追加します。

アドインのインストール

"OdfAddInForWordSetup-en-1.0.exe" ファイルをダブルクリックします。

インストールが正常に完了すると、Microsoft Word で ".ODT" ファイルを開くことができます ([ファイル] メニューに [ODF...] が表示されます)。また、エクスプローラで ".ODT" ファイルを右クリックすると、ドキュメントを Microsoft Word で開くためのオプション ([Microsoft Word で開く] メニュー) が表示されます。

重要 : アドインで開いた ODF ファイルは、読み取り専用の .DOCX ファイルとして Microsoft Word にインポートされます。ファイルを ODF として保存する場合は、まずドキュメントのコピーを作成し (メニューの [名前を付けて保存])、[ODF として保存] メニューを使用します。

トラブルシューティング

Microsoft Word のメニューに [ODF] メニュー項目が表示されない場合は、必要な機能が Word に備わっていないか (「ソフトウェア要件」を参照)、Word のプライマリ相互運用機能アセンブリのインストールに失敗したことが原因として考えられます。そのような場合は、次の手順に従ってアドインを手動で有効にする必要があります。

Microsoft Word XP/2003

  1. [ツール] メニューの [ユーザー設定] をクリックして、[コマンド] タブをクリックします。
  2. [分類] ボックスの [ツール] をクリックします。
  3. [コマンド] ボックスの [COM アドイン] を、[ツール] メニューにドラッグします。[ツール] メニューにメニュー コマンドが表示されたら、[COM アドイン] コマンドを表示させる位置をポイントし、マウスのボタンを放します。
  4. [閉じる] をクリックします。
  5. [ツール] メニューの [COM アドイン] をクリックし、[使用できるアドイン] ボックスで、アドイン名の横のチェック ボックスをオンにします。目的のアドインが一覧に表示されない場合は、[追加] をクリックしてアドインを探し、 [OK] をクリックします。

Microsoft Word 2007

  1. [ファイル] メニューの [Word のオプション] をクリックします。
  2. [アドイン] セクションのボックスの [COM アドイン] をクリックし、[設定] をクリックします。
  3. [ODF Word Add-in] チェック ボックスをオンにし、[OK] をクリックして有効にします。
